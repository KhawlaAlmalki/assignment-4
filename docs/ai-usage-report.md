# **AI Usage Report**

## **Tools Used & Use Cases**

### **ChatGPT (GPT-5)**
Served as a development partner throughout the project by providing explanations, debugging help, and code guidance.

---

## **Stage 1 — Structure & Design**
- Guided creation of the initial HTML/CSS layout using semantic tags and responsive units.
- Helped refine spacing, typography, accessibility, and mobile-first layout.
- Supported organizing core sections: About Me, Experience, Projects, Skills, Awards, Contact.
- Assisted in selecting a balanced color palette and ensuring dark mode compatibility.

---

## **Stage 2 — Interactivity & Logic**
- Helped implement **smooth scroll animations** with `IntersectionObserver`.
- Assisted with the **Projects Section logic**, including:
    - Real-time search
    - Category filtering
    - Sorting (A–Z, Z–A, by date)
- Guided development of the **Experience Timeline** with:
    - Expand/collapse functionality
    - Keyboard accessibility (Enter/Space)
    - ARIA attributes
- Helped build the **Weather Widget**:
    - Real-time temperature
    - Emoji mapping based on conditions
    - Timer using `setInterval`

---

## **Stage 3 — State Management & Persistence**
- Assisted in implementing `localStorage` to persist:
    - Light/dark theme
    - Contact form data
    - Collapsed/expanded experience panels
    - Show/Hide Experience toggle
- Helped refactor JavaScript into cleaner, modular functions.

---

## **Stage 4 — Performance Optimization**
- Helped interpret Lighthouse results and identify slow elements.
- Guided improvements including:
    - Compressing & resizing large images
    - Lazy-loading non-critical images
    - Adding width/height attributes to prevent layout shift
    - Preloading CSS to reduce render-blocking
    - Inlining critical CSS for faster first paint
- Contributed to achieving a **99 Performance Score (Mobile)**.

---

## **Stage 5 — Documentation & Presentation**
- Supported writing and structuring all Markdown documentation.
- Helped refine technical descriptions and ensure clarity.
- Assisted in formatting this AI Usage Report professionally.
